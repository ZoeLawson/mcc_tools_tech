What is MadCap Flare?

MadCap Flare is a content management application used to create, manage, and publish content in a variety of formats and devices.

Why Do Technical Writers Use MadCap Flare?

Here are two reasons, among many, technical writers and documentation teams use MadCap Flare:

  1. Flexibility: Flare has features that make writing content more flexible. For example, you can click-and-drag the XML structure bars (the bars on the left-hand side of the XML Editor) to rearrange the order of content in the document.

  2. Reusable:Single sourcing features like conditions and variables foster content reuse. No longer do you have to manually edit changes to your documentation. Simply use the single source tools available and changes are made automatically.

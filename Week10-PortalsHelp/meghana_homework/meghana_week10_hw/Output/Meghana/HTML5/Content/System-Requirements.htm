System Requirements

The following operating systems are now supported for RSA NetWitness Platform version 11.3:

  • Windows 2019 Server
  • Windows 10 (32 and 64-bit) (up to version 1809)
  • Red Hat Linux 7.x
  • macOS 10.13 (High Sierra)
  • macOS 10.14 (Mojave)

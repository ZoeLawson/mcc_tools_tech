A few terms

There are many more but this should get you started

DPI - Dots per inch. Resolution for print.

PPI - Pixels per inch. Resolution for digital displays.

Raster - Stores information pixel by pixel, color & position, with some compression algorithm.

Vector - Stores images as mathematical expressions.

Lossy - Compression algorithms that lose data

Lossless - Compression algorithms that retain all data

More about raster images

  • BMP — Bitmap. Windows only. Low quality. Avoid.
  • GIF — Supports transparency and animation. Used to be proprietary. Good for line art. Lossy.
  • JPG — Great for photographs. Not so great for diagrams. Lossy.
  • PNG — Supports transparency. Decent for screen captures, diagrams, and photos. Lossless.

Other image types

These are still used, but are rarer these days.

ICO - File format for icons. May contain multiple images in one file.

RAW - Digital camera files.

TIFF - Tag Image File Format. For high quality photographs.

WMF - Windows Meta File. Proprietary Windows image format.

Which type to use

Depends on your output.

Today, PNG is the most universal.

Decent quality for web or PDF, supports transparency, works for screen captures, diagrams, or simple photos.

If you need animation, use GIF.

For actual photographs, use JPG.

SVG can work any where at any resolution. Browser support is relatively recent.

Do more research than I'm giving you.

Graphic terms

A few concepts you should know

Canvas size — the available space available to draw in.

Image size — usually pixels by pixels. The physical size on screen or page will depend on the resolution of the image.

Layer — a container in a graphic for a set of objects

Path — a fancy "line" in a vector application

Drawing terms

These are relatively common across many drawing applications

Align and Distribute - Line up edges or centers, make sure space between objects are equal

Group - Combine disparate objects into a lump that can be moved/copied as one

Snapping or Snap to Grid - Some tools have grids that you can automatically align things to

Bring to Front/Back - Beyond official layers, objects can be on top of one another

Colors

RGB — Colors defined by Red, Green, and Blue levels. Used with light, i.e. monitors.

CYMK — Colors defined by Cyan, Yellow, Magenta, and blacK. Used with print.

For most of tech writing, use RGB colors. They are often defined using Hex codes.

Don't use too many colors.

Make sure your colors are accessible.

Follow any marketing/corporate guidelines.

Editing an image

Tool-agnostic guidelines

  • If at all possible, avoid text in your images. Saves on translation costs.
  • If you must have text, try to use a tool with layers and put all text on the same layer.
Middlesex Community College - Tools and Technologies for Tech Writers - Winter 2023

More image editing

If you have callouts, text, highlighting, blurring, etc.:

  • Reference the source screen capture or have it on a separate layer
  • Do the work in a graphics tool that you can save without "compressing layers"
  • Save the source in the proprietary format and in the output you need

If you do this, when the software changes, it's easier to update the image. You're not starting from scratch.

Middlesex Community College - Tools and Technologies for Tech Writers - Winter 2023

Saving your images

If it's just an icon or screen capture, save as the output format (PNG)

If you are making any sort of changes, save as both a source file such as AI and your output PNG

Usually, save both files with the same name. Exception: If you need different resolutions of the same image for different outputs.

Set the image size and resolution when exporting/saving as the output format.

What size?

This is a really hard question these days.

Monitors used to be pretty standard at 72 dpi or 96 dpi.

Now we have HD monitors, where you have to scale the applications to be legible.

Image sizes on the internet aren't as important. (Unless you're international)

In general, 96 dpi for web/projection and 300 dpi for print mostly works, ish.

Some best practices

Remember, I'm not a graphic designer.

Edit your images in image editing software. Don't rely on the image tools in your authoring tools.

Be precise. Don't have a few extra pixels around your image. Use tools so you have straight (or smooth) lines.

Invest some time learning a graphics editing tool or two to be able to perform basic tasks.

Don't forget your Alt Text.
